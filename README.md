## Commands

```sh
# run local
npm run start

# build docker
docker build --no-cache -t hub-spa .

# run docker
docker run --rm -p 3000:5000 hub-spa

# create project example
nvm install stable --reinstall-packages-from=current
npx create-react-app balkon.hub.spa --template typescript
```
